Prática: Spread Operator e Rest Parameters

Funções Implementadas:

1. Spread em Arrays - unirListas()

Aceita dois arrays e retorna um novo array contendo todos os elementos. 

const frutas = ["Maçã", "Banana"];
const vegetais = ["Cenoura", "Brócolis"];
const todos = unirListas(frutas, vegetais);
// Resultado: ["Maçã", "Banana", "Cenoura", "Brócolis"]

2. Spread em Objetos - atualizarPessoa()

Combina propriedades de dois objetos, retornando um novo objeto.

const pessoa = { nome: "João", idade: 30 };
const atualizacoes = { idade: 31, profissao: "Dev" };
const atualizada = atualizarPessoa(pessoa, atualizacoes);
// Resultado: { nome: "João", idade: 31, profissao: "Dev" }

3. Rest Parameters - mediaNotas()

Aceita número variável de notas e retorna a média.

const media = mediaNotas(8, 7, 9, 10);
// Resultado: 8.5

4. Combinação - exibirResumo()

Exibe um título seguido de todas as informações fornecidas.

exibirResumo("Produto", "Nome: Laptop", "Preço: €899", "Stock: Sim");
// Exibe no console:
// === Produto ===
// - Nome: Laptop
// - Preço: €899
// - Stock: Sim

Instruções:

Abra o Console (F12) para ver todas as demonstrações.
